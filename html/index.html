What is Aflat?

Aflat is designed to provide developers with a high degree of control over their applications while maintaining readability and simplicity. Its minimalistic syntax and low-level features make it suitable for performance-critical applications, and it allows a flexible programming style to suit various projects.

Key Features

  • Lightweight syntax for easy reading and writing
  • Control over memory and efficient resource management
  • Modern constructs without unnecessary complexity
  • Community-driven and actively maintained

Aflat Code Examples

Hello, World!

This is a simple example that prints "Hello, World!" to the console.


import * from "io" under io;
int main() {
    io.print("Hello, World!");
};

Basic Loop

This example demonstrates a loop that iterates 10 times, printing each value.


import {print} from "String" under str;
int main() {
    for mutable int i = 0; i < 10; i = i + 1 {
        str.print(`Value: ${i}\n`);
    };
    return 0;
};
