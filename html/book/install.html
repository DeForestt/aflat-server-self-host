Installing AFlat

AFlat is a Linux-only programming language. Follow the steps below to install it quickly on your system.

One Command to Rule Them All

The fastest way to install AFlat is by running the following command: 

        curl -s https://raw.githubusercontent.com/DeForestt/aflat/main/install.sh | bash

This will download and install the AFlat compiler automatically.

Manual Installation

If you prefer to install AFlat manually, follow these steps:

1. Clone the Repository

        git clone https://github.com/DeForestt/aflat

2. Install Dependencies

Ensure you have `make` installed: 

        sudo apt install make

3. Build AFlat

Navigate into the AFlat directory and compile: 

        cd aflat
        mkdir bin
        make

4. Add AFlat to Your Path

You can either create an alias or add AFlat to your system's PATH.

Option 1: Create an Alias

        echo 'alias aflat="~/aflat/bin/main"' >> ~/.bashrc
        source ~/.bashrc

Option 2: Add to PATH

        echo 'export PATH="$HOME/aflat/bin:$PATH"' >> ~/.bashrc
        source ~/.bashrc

Verifying Installation

To check if AFlat is installed correctly, run: 

        aflat --version

If the version number appears, AFlat is ready to use!
